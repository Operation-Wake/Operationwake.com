# Operation Wake Website v3 Production-ready GitHub package compiled July 2026. ## Included - Refined homepage with crew, mission dashboard, readiness-based journey, and Captain's Log - Final Mission page - Final Journey page organized by capability rather than fixed dates - Final Support page organized around participation before direct funding - Our Story, Videos, and Resources foundations - Shared responsive styling, navigation, logo assets, and site configuration ## Timeline philosophy The public site no longer relies on a fixed departure countdown. Progress is communicated through the current mission phase and readiness milestones. Event-specific countdowns can be added later when a firm, near-term date exists. ## Editing shared information Update `site-config.js` to change the current phase, home port, destination, crew labels, or dashboard progress. ## GitHub / Netlify deployment Upload the extracted contents of this folder to the repository root. Do not upload the ZIP itself. Keep the `assets` folder intact.
